It is a pokemon mobile application developed with React Native.

## Installation

```bash

# If you use npm:

npm install

# and

npx react-native run-android

  

# Or if you use Yarn:

yarn

# and

npx react-native run-android

```

  
  
  

## What Does It Contain?

  

- React Native

- Redux

- PokeAPI
